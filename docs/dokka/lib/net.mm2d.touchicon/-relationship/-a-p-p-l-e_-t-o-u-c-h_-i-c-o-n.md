# APPLE_TOUCH_ICON

`APPLE_TOUCH_ICON`

This expresses that the rel value of the link tag is "apple-touch-icon".

This corresponds to the following description of html.

``` html
<link rel="apple-touch-icon" href="/apple-touch-icon.png" sizes="57x57">
```
