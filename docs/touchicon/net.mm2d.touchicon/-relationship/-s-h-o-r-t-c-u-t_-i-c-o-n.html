SHORTCUT_ICON

SHORTCUT_ICON

This expresses that the rel value of the link tag is "shortcut icon".

This is used only in PageIcon. It is left for old IE, but it is a deprecated description as HTML.

This corresponds to the following description of html.

<link rel="shortcut icon" href="/favicon.ico">
